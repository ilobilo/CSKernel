# CSKernel
C# kernel project template using multiboot and limine

# Building
You will need these programs installed:
* csc (from dotnet)
* xorriso
* ld
* nasm
* qemu-system-x86
* tysila2 (deb package included in source)

Run ```make``` in source root to build and run
